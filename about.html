About this website

I built this website as I wanted a place of my own to highlight my projects, my skills and share some of the things I've learned. It is a static website built from scratch using HTML and CSS, to ease deployment it uses Liquid templates and Front Matter. It is hosted on a private server and the content is served using Jekyll.

Features

Here are some features I've implemented so far:

  • Responsive layout for comfortable reading
  • Dark and light color schemes
  • Alternative text for images
  • Language-dependent syntax highligthing for code snippets
If you're interested and want to know more, take a look at the GitHub repository.

Color schemes

There are two color schemes, one light and one dark. The color schemed served to the visitor depends on the preference provided by their system. Each color scheme consists of six base colors and one CSS style-sheet for code snippets, which have their own colors.
